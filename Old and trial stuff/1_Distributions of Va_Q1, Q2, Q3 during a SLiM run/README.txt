## This folder cntains three main scripts:

### 1_W-F_sim_repeated.slim ###

# This is a SLiM file that runs a Wright-Fisher simulation.
# The simulations runs for 200000 generations
# Stores the full output of the population at every 1000 generations beginning at generation 101000
# The full slim output is stored in "b_Interim_files/SLiM_Outputs"


### 2_Extract_mutations_genomes ###

# This is a python script that depends on the python function [extract_mutations_genomes()] stored in "a_Functions/Function_to_extract_mutations_and_genomes_from_SLiM_output"
# Using the function, this script reads each of the slim outputs generated above and generates the following files
# 1. A text file of all mutations and their information in the entire population ("mutations_[generation number].txt") stored in "b_Interim_files/Mutations"
# 2. A text file of all genomes containing lists of their mutations in the sample drawn at the end of the simulation ("all_genomes_[generation number].txt") stored in "b_Interim_files/Genomes". Note that this file is not used in downstream analyses.
# 3. A csv file containing the c matrix for genomes for a random sample of individuals drawn from the population in the said sample ("c_matrix_[genration number].csv") Stored in "b_Interim_files/C_Matrices". 
# Note that the genomes in the SLiM output are such that genomes 2*i and 2*i - 1 belong to individual i, etc.



### 3_Compute_va_q1_q2_q3 ###

# This is an R script that depends on the function [compute_va_q1_q2_q3()] stored in "a_Functions/Function_to_Compute_V1_Q1_Q2_Q3"
# Using the function and the files generated by the python script, this file computes Va, Q1, Q2, Q3 for each generation and stores these quantities alongside the generation as a .csv file "c_Output/Quantities_BC.csv"
